O Sistema Solar é um projeto interativo desenvolvido em HTML, CSS e JavaScript, criado com o objetivo de estudar animações, transformações e manipulação do DOM.
A ideia inicial era apenas testar o movimentos e animações com css, mas acabei me interessando um pouco mais e projeto evoluiu para uma pequena simulação 
do sitema solar. Durante o desenvolvimento, surgiu o desafio de clicar nos planetas enquanto eles estavam em movimento, o que levou à criação de um botão
de pausa, permitindo que a animação seja interrompida para facilitar a interação e exibição das informações de cada planeta. Além disso, foi adicionado um 
sistema de reversão que retorna a simulação ao estado normal após a observação, criando uma experiência mais fluida e intuitiva. Foram exploradas técnicas 
de animação com @keyframes, transformações com rotate e translate, além da integração entre múltiplos arquivos JavaScript para controlar eventos e exibir 
conteúdos dinâmicos. O design foi pensado para ser leve, educativo e visualmente envolvente, simulando os movimentos reais dos corpos celestes em torno do Sol.
O projeto foi criado em caráter de estudo, com o objetivo de aprimorar habilidades de animação, lógica e interatividade. Futuramente, há planos de adicionar 
zoom nos planetas, foco automático, mais informações científicas e até trilha sonora ambiente para tornar a simulação ainda mais imersiva.
